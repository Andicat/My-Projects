Расстановка ферзей

"Расстановка ферзей" - такая расстановка 8 ферзей на шахматной доске (8х8), при которой ни один из ферзей не может сбить ни одного другого ферзя. Ферзь бьёт в любом направлении - горизонтальном, вертикальном, диагональном. Найдем все возможные расстановки, дадим возможность пользователю ввести номер расстановки. Отобразим шахматную доску с выбранной расстановкой - расставленными ферзями. При нажатии мышью на любого из ферзей - подсветим клетки доски, которые он "пробивает".

Обход конем шахматной доски

Найдем программно один способ обойти все 64 клетки доски конём, чтобы в каждой клетке конь побывал ровно 1 раз. Обход должен начинаться с любой угловой клетки. Отобразим найденный способ (в каждой клеточке - номер хода, на котором конь попадает в эту клеточку). По нажатию кнопки - отобразим процесс обхода конём доски в динамике. Клетки, в которых конь уже побывал, подсветим.

Лифт

Построим программно дом, лифт (количество этажей дома настраивается программно). На каждом этаже - две кнопки для вызова лифта, "вверх" и "вниз", которые подсвечиваются, если нажаты мышью. В лифте - кнопки с номерами этажей, которые подсвечиваются, если нажаты мышью. Одновременно может быть нажато несколько кнопок на разных этажах и несколько кнопок внутри лифта. Добавим пасссажиров и запустим лифт.
